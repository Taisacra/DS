Formulários HTML

01.01 Elementos de formulários

Elementos input

  • O elemento HTML input é o elemento de formulário mais usado.
  • O elemento input pode ser exibido de várias maneiras, dependendo do tipo de atributo associado a ele.

Aqui estão alguns exemplos:

Exemplo de Formulários



  • <form action="/action_page.php">:
    Esta linha define um formulário HTML. O atributo action especifica para onde os dados do formulário serão enviados quando o usuário clicar no botão "Submit". Neste caso, os dados seriam enviados para "/action_page.php".

  • <label for="fname">First name:</label><br>:
    Este trecho cria uma etiqueta de rótulo para o campo de entrada de texto do primeiro nome. O atributo for está associado ao ID do campo de entrada correspondente, que é "fname". A quebra de linha <br> é usada para adicionar uma quebra de linha após o rótulo.

  • <input type="text" id="fname" name="fname" value="John"><br>:
    Aqui, é criado um campo de entrada de texto para o primeiro nome. O tipo é definido como "text", o ID como "fname" e o nome como "fname". O valor inicial é definido como "John". Após o campo de entrada, há uma quebra de linha.

  • <label for="lname">Last name:</label><br>:
    Semelhante ao primeiro rótulo, isso cria um rótulo para o campo de entrada do sobrenome, associado ao ID "lname". Uma quebra de linha <br> é adicionada para separar visualmente.

  • <input type="text" id="lname" name="lname" value="Doe"><br><br>:
    Este é o campo de entrada de texto para o sobrenome. O tipo é "text", o ID é "lname", o nome é "lname" e o valor inicial é "Doe". Duas quebras de linha são usadas para adicionar espaço.

  • <input type="submit" value="Submit">:
    Este é o botão de envio do formulário. Ele é do tipo "submit", o que significa que quando clicado, o formulário será enviado para o URL especificado no atributo "action". O texto exibido no botão é "Submit".

  • </form>:
    Fecha o formulário HTML.

O label representa uma legenda para um item em uma interface de usuário.
